# PoO Demo

## Pre-requiesite

A CodeChain node with solo chain. // we use solo's genesis address to pay for the fee.

## Setup

Create public key, private key, asset.

```bash
yarn run-setup
```

## Run server

```bash
yarn run-server
```

## Run client

Change parameters in the src/client.ts file using printed values from the setup.

```bash
yarn run-client
```
